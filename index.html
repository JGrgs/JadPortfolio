Maintain-Me

React | JavaScript | Node.js | Socket.io | Express | MongoDB/Mongoose

Maintain-Me is an app created to help you track you vehicle's maintenance schedules. Every user gets to add their vehicle/vehicles into their profile and add their maintenance schedules into a table providing the maintenance type, due date/due at (miles). With the help of google map react, the google maps geolocation API and yelp's API, Maintain-Me renders a map pinpointing all the mechanics around the user based on the user's location which is provided to us when a user signs up.
